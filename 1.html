HTML

html은 전체적으로 head와 body로 나뉜다.

앞에있는 것을 열린 태그, 뒤에있는 것을 닫힌 태그라고 한다. 닫힌 태그는 /를 붙인다.

li 태그 : list의 앞 두자리, li와 같은 항목들은 어디서부터 어디까지 연관된 항목인지 경계를 짓기 위한 부모 태그가 필요
ul,ol 태그 : li 태그의 부모 태그, ul,ol의 자식태그는 li 태그, ol은 ordered list, ul은 unoredered list이다.

br 태그 : 줄바꿈을 나타내는 시각적인 의미만 가지고 있기에 무언가를 감쌀 필요가 없다
p 태그 : 단락(paragraph)이 바뀌는데 새로운 줄을 나타내면서 줄간격이 달라짐 br은 단지 줄바꿈이지만 p는 어디서 부터 어디까지 단락이다 라는 것을 표현함, css를 통해서 훨씬 정교하게 사용할 수 있다.

h1 ~ h6 태그 : 제목(head)를 나타낸다. h1이 가장 큰 제목이고 h6으로 갈 수록 폰트 크기가 작아짐
태그의 이름만으로는 정보가 부족하다 -> 새로운 문법 출현
어떤 이미지인지를 알려주기 위한 속성(Attribute) 등장. 위치 상관없음. 태그의 이름만으로는 정보가 부족할 때 사용


a 태그 : 링크(웹페이지와 웹페이지를 이어주는 태그)
- 배가 닻을 내리듯 다른 웹페이지에 anchor를 내리는 것. 이 때 어디다 닻을 내려야 하는지 설명해주는 속성 href(hypertext reference)를 사용해 링크를 검.
target 속성을 이용하면 빈 탭에서 링크가 열리게 할 수 있으며 title 속성을 달아주면 링크에 대한 설명을 툴팁으로 띄울 수 있다.

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.
strong 태그 : bold u 태그 : underline

title 태그 : 브라우저 탭에 보이는 웹페이지의 제목을 만들어준다.
meat 태그 : 웹 브라우저보고 utf-8 방식으로 읽어라! 라고 명령
-> utf-8 방식으로 저장한 파일 : 웹 브라우저가 utf-8로 열어줘야 깨지는 문자 없이 잘 표현됨.
doctype : 그냥 관용표현
